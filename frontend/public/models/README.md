## Better free models

Download `.glb` files and rename them to match the table above:

- [Khronos glTF Sample Assets](https://github.khronos.org/glTF-Assets/) — CC0/BY models
- [Poly Pizza](https://poly.pizza/) — free low-poly models
- [Sketchfab](https://sketchfab.com/) — filter by **Downloadable** + **CC** license

After replacing files, refresh the page — no code changes needed.
